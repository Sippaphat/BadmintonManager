# Badminton Manager Web

This is a React web application ported from the React Native Badminton Manager Mobile app.

## Project Structure

- `src/App.jsx`: Main application logic (ported from `App.js`).
- `src/App.css`: Stylesheet (ported from `StyleSheet` in React Native).
- `src/main.jsx`: Entry point.

## How to Run

1.  Open a terminal in this directory (`BadmintonManager`).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the link shown in the terminal (usually `http://localhost:5173`) in your browser.

## Features

-   Manage players (Add/Edit/Remove).
-   Set up number of courts.
-   Auto-assign matches based on play count fairness.
-   Track game status and finish matches.
